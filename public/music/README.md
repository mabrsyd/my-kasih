This directory should contain your romantic instrumental background music.

Required file: romantic-instrumental.mp3

To add your music:

1. Place your MP3 file in this directory
2. Name it: romantic-instrumental.mp3
3. The file will be auto-played by the MusicPlayer component

Recommended: Use royalty-free romantic instrumental music (min 2-3 minutes for looping)

Good sources:

- YouTube Audio Library (free)
- Incompetech (free with attribution)
- Pixabay Music (free)
- Pexels Music (free)

The music player has fallback behavior if auto-play fails on some browsers.
